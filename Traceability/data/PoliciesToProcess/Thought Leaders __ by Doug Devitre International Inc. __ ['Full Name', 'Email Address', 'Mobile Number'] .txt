web cookies and beacons
web cookies (also known as http cookies, or just cookies) are parcels of text that are sent by a server to a web browser, and then sent back unchanged by the browser each time it accesses that server
in other words, a cookie is a piece of data stored on a user’s computer that is tied to information about the user
web beacons are objects that are embedded in a webpage or email that are invisible to the user, but allow checking that a user has viewed the page or email
other names are web bug, tracking bug, tag or page tag
