use of "cookies" visitors to the ewtn web site should be aware that non-personal information and data is automatically collected by the ewtn web site through the use of "cookies
" "cookies" are small text files which a web site can use to recognize repeat visitors, facilitate a visitor's ongoing access to, and use of, the site, and allow a web site to track usage behavior and compile data that will allow content improvements
if a visitor does not want information collected through the use of cookies, there is a simple procedure in most browsers that allows the visitor to deny or accept the cookie feature
ewtn uses "cookie" technology only to obtain non-personal information from its online visitors in order to improve visitors' online experience and facilitate their visit of ewtn's web site
video players and country selection are saved as an anonymous cookie that stores the region you selected when visiting this website
